Class 4: Total Derivative (Differentials)

In this class, we explore the concept of total derivative (differentials), combining partial derivatives to approximate changes in a function due to multivariate inputs. This allows us to predict how small changes in inputs affect the output.

From Partial Derivatives to Total Derivative

The partial derivatives $f_x$ and $f_y$ capture how $f$ changes when moving purely in the $x$ or $y$ direction. However, if both $x$ and $y$ change simultaneously, the overall change in $f$ is described by the total derivative.

Total Differential

For a differentiable function $z=f(x,y)$, the total differential $dz$ combines the effects of changes in both variables:

$$dz = \frac{\partial f}{\partial x}(x,y)\,dx + \frac{\partial f}{\partial y}(x,y)\,dy = f_x(x,y)\,dx + f_y(x,y)\,dy$$

This formula originates from the linear approximation (tangent plane) we studied in Class 3. It indicates that for small changes $dx, dy$, we have:

$$\Delta z \approx f_x(x,y)\,\Delta x + f_y(x,y)\,\Delta y$$

Geometric Interpretation

In interpretation, $dz$ is the height change one would predict by "following the tangent plane." If $(dx,dy)$ is a small step in the $xy$-plane, then $dz$ is the corresponding change in $z$ along the tangent plane.

The total derivative at a point $(a,b)$ can be understood as the linear map $(dx,dy) \mapsto f_x(a,b)\,dx + f_y(a,b)\,dy$.

In vector form, if $\mathbf{u} = \langle dx, dy\rangle$, then:

$$dz = \nabla f(a,b)\cdot \mathbf{u}$$

where $\nabla f = \langle f_x,\;f_y\rangle$ is the gradient vector.

Conditions for Total Differentiability

It is important to note that a function $f(x,y)$ may have partial derivatives without being fully differentiable (meaning the linear approximation might not perfectly describe small changes from that point).

However, if $f_x$ and $f_y$ exist and are continuous in a neighborhood of $(a,b)$, then $f$ is differentiable at $(a,b)$, and the total derivative exists. In such cases, the linear approximation is accurate for infinitesimal changes.

Example 4.1

Use differentials to approximate the change in $z$ for $z=\sqrt{x^2+y^2}$ when $(x,y)$ moves from $(3,4)$ to $(3.1, 4.2)$.

Solution:

Here $f(x,y) = \sqrt{x^2+y^2}$ represents the distance from the origin to the point $(x,y)$.

First, we compute the partial derivatives:

\begin{align*} f_x &= \frac{\partial}{\partial x}\sqrt{x^2+y^2} = \frac{x}{\sqrt{x^2+y^2}}\\ f_y &= \frac{\partial}{\partial y}\sqrt{x^2+y^2} = \frac{y}{\sqrt{x^2+y^2}} \end{align*}

At the point $(3,4)$, these evaluate to:

\begin{align*} f_x(3,4) &= \frac{3}{\sqrt{3^2+4^2}} = \frac{3}{5}\\ f_y(3,4) &= \frac{4}{\sqrt{3^2+4^2}} = \frac{4}{5} \end{align*}

Note that $\sqrt{3^2+4^2} = 5$, which is the distance from the origin to the point $(3,4)$.

The differentials are:

\begin{align*} dx &= 3.1 - 3 = 0.1\\ dy &= 4.2 - 4 = 0.2 \end{align*}

Using the formula for the total differential:

\begin{align*} dz &\approx f_x(3,4)\,dx + f_y(3,4)\,dy\\ &= \frac{3}{5}(0.1) + \frac{4}{5}(0.2)\\ &= 0.03 + 0.16\\ &= 0.19 \end{align*}

We predict $z$ will increase by approximately $0.19$ units.

To verify our approximation, we calculate the actual change:

\begin{align*} z \text{ at } (3.1,4.2) &= \sqrt{3.1^2+4.2^2}\\ &= \sqrt{9.61 + 17.64}\\ &= \sqrt{27.25}\\ &\approx 5.22 \end{align*}

The original value was $z = \sqrt{3^2+4^2} = 5$

Therefore, the actual change is:

\begin{align*} \Delta z &= 5.22 - 5\\ &= 0.22 \end{align*}

Our differential approximation of $0.19$ is very close to the actual change of $0.22$, with an error of only $0.03$. This demonstrates the effectiveness of the differential approach for small changes in the input variables.

Comparison: Partial vs. Total Derivatives

Aspect Partial Derivatives Total Derivative
Definition Rate of change in one variable direction Combined effect of changes in all variables
Variables All other variables held constant All variables allowed to vary
Notation $\frac{\partial f}{\partial x}$ or $f_x$ $df$ or $\frac{df}{dt}$ (if variables depend on $t$)
Geometric Slope in coordinate directions Linear approximation in any direction
Application Finding tangent planes Approximating function changes

Key Insights about Total Derivatives

  • The total derivative combines partial derivatives to predict changes when multiple variables change simultaneously.
  • The formula for the total differential, $dz = f_x\,dx + f_y\,dy$, emerges naturally from the tangent plane approximation.
  • For a function to be totally differentiable, the partial derivatives must exist and be continuous in a neighborhood of the point.
  • The gradient vector $\nabla f = \langle f_x, f_y \rangle$ encapsulates the total derivative's behavior in a coordinate-free way.
  • In engineering applications, differentials provide a powerful tool for estimating how small changes in inputs affect outputs, essential for sensitivity analysis and error propagation studies.
