Class 7: Local Extrema of Multivariate Functions

In this class, we will explore how to find and characterize local extrema (maxima and minima) of functions of several variables, extending the concepts from single-variable calculus to multiple dimensions.

Local Extrema in Multiple Variables

For a function $f: \mathbb{R}^n \to \mathbb{R}$, a point $\mathbf{p} \in \mathbb{R}^n$ is a:

  • Local maximum if there exists a neighborhood $U$ of $\mathbf{p}$ such that $f(\mathbf{x}) \leq f(\mathbf{p})$ for all $\mathbf{x} \in U$.
  • Local minimum if there exists a neighborhood $U$ of $\mathbf{p}$ such that $f(\mathbf{x}) \geq f(\mathbf{p})$ for all $\mathbf{x} \in U$.
  • Local extremum if it is either a local maximum or a local minimum.

Critical Points

A critical step in finding extrema is identifying critical points. For a differentiable function $f: \mathbb{R}^n \to \mathbb{R}$, a point $\mathbf{p}$ is a critical point if the gradient of $f$ at $\mathbf{p}$ is zero:

$$\nabla f(\mathbf{p}) = \mathbf{0}$$

For a function $f(x,y)$ of two variables, this means that both partial derivatives must be zero at the critical point:

$$\frac{\partial f}{\partial x}(x_0, y_0) = 0 \quad \text{and} \quad \frac{\partial f}{\partial y}(x_0, y_0) = 0$$

Just like in single-variable calculus, a critical point is a necessary condition for a local extremum (if the function is differentiable at that point), but it's not sufficient. A critical point could be a local maximum, a local minimum, or neither (a saddle point).

Example 7.1

Question: Find the critical points of $f(x,y) = x^2 + y^2 - 2x - 4y + 5$.

Solution:

We compute the partial derivatives:

$$\frac{\partial f}{\partial x} = 2x - 2$$ $$\frac{\partial f}{\partial y} = 2y - 4$$

Setting these equal to zero:

$$2x - 2 = 0 \implies x = 1$$ $$2y - 4 = 0 \implies y = 2$$

Thus, the only critical point is $(1, 2)$.

Saddle Points

Unlike in single-variable calculus, multivariate functions can have critical points that are neither maxima nor minima. These are called saddle points.

A saddle point is a critical point where the function increases in some directions and decreases in others. The classic example is the function $f(x,y) = x^2 - y^2$, which has a saddle point at the origin $(0,0)$. If you fix $y=0$ and vary $x$, the function behaves like $x^2$ (a minimum at $x=0$). But if you fix $x=0$ and vary $y$, the function behaves like $-y^2$ (a maximum at $y=0$).

Figure 7.1: Visualization of a saddle point for the function f(x,y) = x² - y². The surface rises in the x-direction but falls in the y-direction around the origin.
Example 7.2

Question: Find the critical points of $f(x,y) = x^2 - y^2 + 2x$ and determine their nature.

Solution:

We compute the partial derivatives:

$$\frac{\partial f}{\partial x} = 2x + 2$$ $$\frac{\partial f}{\partial y} = -2y$$

Setting these equal to zero:

$$2x + 2 = 0 \implies x = -1$$ $$-2y = 0 \implies y = 0$$

Thus, the only critical point is $(-1, 0)$.

To determine the nature of this critical point, we need to analyze the behavior of the function near this point. Intuitively, if we fix $y=0$ and vary $x$, the function becomes $f(x,0) = x^2 + 2x$, which has a minimum at $x=-1$. However, if we fix $x=-1$ and vary $y$, the function becomes $f(-1,y) = 1 - y^2$, which has a maximum at $y=0$. Since the function increases in some directions and decreases in others around $(-1, 0)$, it is a saddle point.

Types of Critical Points

For a function $f: \mathbb{R}^2 \to \mathbb{R}$ with a critical point at $(x_0, y_0)$, we can classify it based on the behavior of the function around this point:

  • Local maximum: The function decreases in all directions from the critical point.
  • Local minimum: The function increases in all directions from the critical point.
  • Saddle point: The function increases in some directions and decreases in others from the critical point.

In the next class, we will develop a systematic approach to classify critical points using the second derivatives of the function, leading to the second derivative test for functions of several variables.

Geometric Interpretation

For a function $f(x,y)$, we can visualize critical points by looking at the graph $z = f(x,y)$ in three-dimensional space:

  • A local maximum corresponds to a "peak" on the surface.
  • A local minimum corresponds to a "valley" on the surface.
  • A saddle point is similar to a mountain pass, where the surface rises in one direction but falls in another.

Key Insights about Critical Points

  • Critical points occur where all partial derivatives are zero.
  • Critical points can be maxima, minima, or saddle points.
  • Multivariate functions often have saddle points, which are neither maxima nor minima.
  • The second derivative test (which we'll cover next) provides a systematic way to classify critical points.
  • Finding critical points is central to many engineering applications, including optimization problems, stability analysis, and potential energy configurations.
