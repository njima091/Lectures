微分方程式講義ノート

第1節：解法の基本

主な解法一覧：

手法特徴
変数分離法初等的で直接的
定数変化法非斉次方程式に有効
級数展開法解が初等関数で表せない場合に使用
演算子法定係数線形方程式に有効

第2節：2階線形同次微分方程式

一般形と特性方程式

$$y'' + ay' + by = 0$$

対応する特性方程式：

$$r^2 + ar + b = 0$$

判別式による分類と解

実数重解 $D = 0$

$$y = (C_1 + C_2x)e^{rx}$$

異なる実数解 $D > 0$

$$y = C_1e^{r_1x} + C_2e^{r_2x}$$

複素共役解 $D < 0$

$$y = e^{\alpha x}(C_1\cos(\beta x) + C_2\sin(\beta x))$$

第3節：微分演算子法

定義と表現

$D = \frac{d}{dx}$ とすると、$y'' + ay' + by = 0$ は：

$$(D^2 + aD + b)y = 0$$

手順

  1. 演算子形式に変換
  2. 演算子多項式の因数分解
  3. 順に作用させて解を得る

例：$y'' - 3y' + 2y = 0$

演算子形式：$(D - 1)(D - 2)y = 0$

解：

$$y = C_1e^x + C_2e^{2x}$$

非同次方程式への応用：$y'' + ay' + by = f(x)$

演算子形式：$(D^2 + aD + b)y = f(x)$

解の構成：

  • 同次方程式の一般解 $y_h$ を求める
  • 特解 $y_p$ を適当な方法（未定係数法など）で求める
  • 一般解は $y = y_h + y_p$

例：$y'' - 3y' + 2y = e^x$

演算子形式：$(D - 1)(D - 2)y = e^x$

右辺 $e^x$ は $D = 1$ に対応するので、$(D - 2)y = Ae^x$ と仮定し：

$$y_p = \frac{Ae^x}{D - 2} = \frac{e^x}{-1} = -e^x$$

同次解 $y_h = C_1e^x + C_2e^{2x}$ と合わせて：

$$y = C_1e^x + C_2e^{2x} - e^x$$

利点

  • 代数的操作が可能
  • 高次方程式にも応用可
  • 体系的な解法

第4節：応用例

バネ-質量系

$$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = 0$$

$m$: 質量, $c$: 減衰係数, $k$: バネ定数

LRC回路

$$L\frac{d^2i}{dt^2} + R\frac{di}{dt} + \frac{1}{C}i = 0$$

$L$: インダクタンス, $R$: 抵抗, $C$: 静電容量

第5節：まとめ

判別式 $D = a^2 - 4b$ により解の型が決定：

  • $D > 0$: $y = C_1e^{r_1x} + C_2e^{r_2x}$
  • $D = 0$: $y = (C_1 + C_2x)e^{rx}$
  • $D < 0$: $y = e^{\alpha x}(C_1\cos(\beta x) + C_2\sin(\beta x))$
