第1節：微分方程式の解法

微分方程式の主な解法：

解法 簡単 難しい
1) 変数分離法 ○ ×
2) 定数変化法 × ○
3) 級数法 × ○
4) 微分演算子法 × ○

第2節：変数分離法

変数分離形の微分方程式：

$P(y) y' = Q(x)$

この形式では、yに関する関数と微分がすべて左辺に、xに関する関数がすべて右辺にあります。

解法手順：

  1. 方程式を \(P(y) \frac{dy}{dx} = Q(x)\) の形に書き換える
  2. 両辺を変数ごとに分離する： \(P(y) dy = Q(x) dx\)
  3. 両辺を積分する： \(\int P(y) dy = \int Q(x) dx + C\)
  4. 積分を実行し、可能であればyについて解く

第3節：定数変化法

一階線形微分方程式：

$y' + P(x)y = Q(x)$

この形式は積分因子を用いて解くことができます。

解法手順：

  1. 積分因子を求める： \(\mu(x) = e^{\int P(x) \, dx}\)
  2. 両辺に \(\mu(x)\) をかける： \(\mu(x)y' + \mu(x)P(x)y = \mu(x)Q(x)\)
  3. 左辺は次のように変形できる： \(\frac{d}{dx}(\mu(x)y) = \mu(x)Q(x)\)
  4. 両辺を積分する： \(\mu(x)y = \int \mu(x)Q(x) \, dx + C\)
  5. yについて解く： \(y = \frac{1}{\mu(x)}(\int \mu(x)Q(x) \, dx + C)\)

$$\log y = \int \cos x \, dx - \frac{x^2}{2} + C$$

例題1：変数分離法の例

$y' = \frac{x}{y}$

解法：

  1. 変数を分離する形に書き換える： \(y \cdot y' = x\)
  2. 両辺を積分変数の形に： \(y \, dy = x \, dx\)
  3. 両辺を積分： \(\int y \, dy = \int x \, dx\)
  4. \(\frac{y^2}{2} = \frac{x^2}{2} + C\)
  5. 最終解： \(y^2 = x^2 + 2C\) または \(y = \pm\sqrt{x^2 + C'}\) （ここで \(C' = 2C\)）

例題2：定数変化法の例

$$y' + xy = x$$

解法手順：

  1. 積分因子を求める： \(e^{\int x \, dx} = e^{\frac{x^2}{2}}\)
  2. 両辺に積分因子をかける： \(e^{\frac{x^2}{2}}y' + xe^{\frac{x^2}{2}}y = xe^{\frac{x^2}{2}}\)
  3. これは次のように変形できる： \(\frac{d}{dx}(e^{\frac{x^2}{2}}y) = xe^{\frac{x^2}{2}}\)
  4. 両辺を積分する： \(e^{\frac{x^2}{2}}y = \int xe^{\frac{x^2}{2}} \, dx = e^{\frac{x^2}{2}} + C\)
  5. 最終解： \(y = 1 + Ce^{-\frac{x^2}{2}}\)

例題5

$$y' - y = e^x$$

解法手順：

  1. 積分因子： \(e^{\int -1 \, dx} = e^{-x}\)
  2. 両辺をかける： \(e^{-x}y' - e^{-x}y = 1\)
  3. これは簡略化して： \(\frac{d}{dx}(e^{-x}y) = 1\)
  4. 積分： \(e^{-x}y = x + C\)
  5. 最終解： \(y = (x + C)e^x\)

例題6

$$y' + xy = 0$$

解法手順：

  1. 積分因子： \(e^{\int x \, dx} = e^{\frac{x^2}{2}}\)
  2. 両辺をかける： \(e^{\frac{x^2}{2}}y' + xe^{\frac{x^2}{2}}y = 0\)
  3. これは次のようになる： \(\frac{d}{dx}(e^{\frac{x^2}{2}}y) = 0\)
  4. 積分： \(e^{\frac{x^2}{2}}y = C\)
  5. 最終解： \(y = Ce^{-\frac{x^2}{2}}\)
