第11回　数列の極限 (ε–N 論法)

1. 数列の収束の定義

定義 (収束)
数列 $\{a_n\}$ が実数 $a$ に収束するとは，次が成り立つことをいう：

$$ \forall \varepsilon > 0, \; \exists N \in \mathbb{N} \text{ s.t. } \forall n \in \mathbb{N}, \; n > N \implies |a_n - a| < \varepsilon. $$

このとき，$\displaystyle \lim_{n\to\infty} a_n = a$ または $a_n \to a \; (n \to \infty)$ と書く。$a$ を数列 $\{a_n\}$ の極限値という。

(定義の解釈) どんなに小さな正の数 $\varepsilon$ を指定されても，その $\varepsilon$ に応じて十分大きな番号 $N$ を見つけることができる。そして，その $N$ より先の項 ($n > N$) はすべて，極限値 $a$ との差が $\varepsilon$ 未満になる。つまり，添え字 $n$ を大きくすれば，$a_n$ はいくらでも $a$ に近づくことを意味する。

2. ε-N論法による証明の型

ε-N論法に慣れるまでは、証明の「型」を覚えるのが有効である。ここでは、よく使われる2つのパターンを紹介する。

パターン1：$\lim a_n = a \implies \lim b_n = b$ の証明

  1. 目標の確認：示すべきは $\lim b_n = b$、すなわち $\forall \varepsilon > 0, \exists N \in \mathbb{N} \text{ s.t. } n>N \implies |b_n - b| < \varepsilon$ である。
  2. 任意に $\varepsilon > 0$ をとる：証明は「任意の正の数 $\varepsilon$ に対して...」という文言で始める。
  3. 仮定の利用：仮定 $\lim a_n = a$ が使える。これは，$\forall \varepsilon' > 0$ に対して，ある $N_1$ が存在し，$n > N_1 \implies |a_n - a| < \varepsilon'$ が成り立つことを意味する。
  4. ε と ε' の関係式：ゴールである $|b_n - b| < \varepsilon$ を導くために、都合の良い $\varepsilon'$ を設定する。例えば $\varepsilon' = \varepsilon/2$ や $\varepsilon' = \frac{\varepsilon}{C}$ のように、$\varepsilon$ の式で表すことが多い。
  5. N の決定：ステップ3で存在が保証された $N_1$ を使って、$N$ を定める。$N=N_1$ とすることもあれば、他の条件と合わせて $N=\max\{N_1, N_2\}$ のようにとることもある。
  6. 結論：$n > N$ ならば、一連の不等式評価によって $|b_n - b| < \varepsilon$ が示されることを記述する。

Q. なぜ仮定の方の $\varepsilon$ を $\varepsilon'$ や $\varepsilon/2$ に勝手に変えて良いのか？
A. 収束の定義は「任意の（どんな）」正の数に対して成り立つことを保証しているからである。$\varepsilon$ が正の数なら、$\varepsilon/2$ や $\varepsilon/C$ (Cは正の定数) もまた正の数である。したがって、$\varepsilon/2$ という正の数に対して定義を適用し、対応する番号 $N_1$ を見つけてくることができる。証明のゴールである「$\varepsilon$」と区別するために、仮定で使う値を便宜上 $\varepsilon'$ と書くこともある。

3. 極限の基本性質とその証明

定理 (極限の演算)
数列 $\{a_n\}$, $\{b_n\}$ がそれぞれ $a, b$ に収束するとき、次が成り立つ。
  1. 和： $\displaystyle \lim_{n\to\infty} (a_n + b_n) = a + b$
  2. 積： $\displaystyle \lim_{n\to\infty} (a_n b_n) = ab$
  3. 商： $b \neq 0$ かつ全ての $n$ で $b_n \neq 0$ ならば， $\displaystyle \lim_{n\to\infty} \frac{a_n}{b_n} = \frac{a}{b}$

(1) 和の極限の証明

任意の $\varepsilon > 0$ をとる。 仮定より、$a_n \to a$ なので、$\varepsilon/2 > 0$ に対して、ある自然数 $N_1$ が存在して、$n > N_1 \implies |a_n - a| < \varepsilon/2$ が成り立つ。 同様に、$b_n \to b$ なので、$\varepsilon/2 > 0$ に対して、ある自然数 $N_2$ が存在して、$n > N_2 \implies |b_n - b| < \varepsilon/2$ が成り立つ。 ここで $N = \max\{N_1, N_2\}$ とおく。すると、$n > N$ ならば $n > N_1$ かつ $n > N_2$ なので、三角不等式より、 $$ |(a_n + b_n) - (a + b)| = |(a_n - a) + (b_n - b)| \le |a_n - a| + |b_n - b| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. $$ よって $\displaystyle \lim_{n\to\infty} (a_n + b_n) = a + b$ が示された。

(2) 積の極限の証明

収束する数列は有界である（後述の定理より）。よって、ある $M > 0$ が存在して、すべての $n$ について $|a_n| \le M$ となる。 任意の $\varepsilon > 0$ をとる。 $a_n \to a$ より、$\frac{\varepsilon}{2(|b|+1)} > 0$ に対して、ある $N_1$ が存在し、$n>N_1 \implies |a_n - a| < \frac{\varepsilon}{2(|b|+1)}$ とできる。 $b_n \to b$ より、$\frac{\varepsilon}{2M} > 0$ に対して、ある $N_2$ が存在し、$n>N_2 \implies |b_n - b| < \frac{\varepsilon}{2M}$ とできる。 $N = \max\{N_1, N_2\}$ とおく。$n > N$ ならば、 $$ \begin{aligned} |a_n b_n - ab| &= |a_n b_n - a_n b + a_n b - ab| \\ &= |a_n(b_n - b) + b(a_n - a)| \\ &\le |a_n||b_n - b| + |b||a_n - a| \\ &< M \cdot \frac{\varepsilon}{2M} + |b| \cdot \frac{\varepsilon}{2(|b|+1)} \\ &< \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. \end{aligned} $$ よって $\displaystyle \lim_{n\to\infty} (a_n b_n) = ab$ が示された。

(3) 商の極限の証明

まず $\displaystyle \lim_{n\to\infty} \frac{1}{b_n} = \frac{1}{b}$ を示す。$b \neq 0$ なので $|b|>0$ である。 $b_n \to b$ より、$\frac{|b|}{2} > 0$ に対してある $N_1$ が存在し、$n>N_1 \implies |b_n - b| < \frac{|b|}{2}$。 このとき、$|b| = |b-b_n+b_n| \le |b-b_n| + |b_n| < \frac{|b|}{2} + |b_n|$。よって $|b_n| > |b| - \frac{|b|}{2} = \frac{|b|}{2}$。 任意の $\varepsilon > 0$ をとる。$b_n \to b$ より、$\frac{|b|^2 \varepsilon}{2} > 0$ に対してある $N_2$ が存在し、$n>N_2 \implies |b_n - b| < \frac{|b|^2 \varepsilon}{2}$。 $N = \max\{N_1, N_2\}$ とおく。$n > N$ ならば、 $$ \left|\frac{1}{b_n} - \frac{1}{b}\right| = \frac{|b - b_n|}{|b_n||b|} < \frac{|b-b_n|}{(|b|/2)|b|} = \frac{2|b-b_n|}{|b|^2} < \frac{2}{|b|^2} \cdot \frac{|b|^2 \varepsilon}{2} = \varepsilon. $$ これで $\displaystyle \lim_{n\to\infty} \frac{1}{b_n} = \frac{1}{b}$ が示された。あとは(2)の積の極限の結果を用いれば、$\displaystyle \lim_{n\to\infty} \frac{a_n}{b_n} = \lim_{n\to\infty} a_n \cdot \frac{1}{b_n} = a \cdot \frac{1}{b} = \frac{a}{b}$ を得る。

4. 単調数列と収束

定義 (単調列)
  • すべての $n \in \mathbb{N}$ で $a_n \le a_{n+1}$ が成り立つとき、数列 $\{a_n\}$ は単調増加であるという。
  • すべての $n \in \mathbb{N}$ で $a_n \ge a_{n+1}$ が成り立つとき、数列 $\{a_n\}$ は単調減少であるという。
単調増加または単調減少の数列を単調列という。
定理 (有界単調列の収束)
上に有界な単調増加数列は収束し、その極限値は上限 (sup) に等しい。
数列 $\{a_n\}$ は単調増加で、集合 $A = \{a_n \mid n \in \mathbb{N}\}$ は上に有界であるとする。 実数の連続性（ワイエルシュトラスの上限公理）により、上限 $\alpha = \sup A$ が存在する。この $\alpha$ が極限値であることを示す。 任意の $\varepsilon > 0$ をとる。上限の定義より、$\alpha - \varepsilon$ は $A$ の上界ではない。 したがって、$\alpha - \varepsilon < a_N$ を満たすような項 $a_N$ が存在する。 $\{a_n\}$ は単調増加なので、$n > N$ である任意の $n$ に対して $a_N \le a_n$ が成り立つ。 また、$\alpha$ は上界なので、すべての $n$ に対して $a_n \le \alpha$ である。 以上をまとめると、$n > N$ なるすべての $n$ に対して $$ \alpha - \varepsilon < a_N \le a_n \le \alpha < \alpha + \varepsilon $$ が成り立つ。これは $|a_n - \alpha| < \varepsilon$ を意味する。 よって、$\displaystyle \lim_{n\to\infty} a_n = \alpha = \sup A$ が示された。
■

5. 部分列とボルツァーノ・ワイエルシュトラスの定理

定義 (部分列)
自然数の狭義単調増加列 $n_1 < n_2 < n_3 < \dots$ を考える。このとき、数列 $\{a_{n_k}\}_{k=1}^\infty$ を、元の数列 $\{a_n\}$ の部分列という。
$a_n = (-1)^n$ とする。
  • $n_k = 2k$ ととると、部分列 $\{a_{2k}\} = \{(-1)^{2k}\} = \{1, 1, 1, \dots\}$ を得る。これは 1 に収束する。
  • $n_k = 2k-1$ ととると、部分列 $\{a_{2k-1}\} = \{(-1)^{2k-1}\} = \{-1, -1, -1, \dots\}$ を得る。これは -1 に収束する。
数列 $\{a_n\}$ が $a$ に収束するならば、その任意の部分列 $\{a_{n_k}\}$ も同じ極限値 $a$ に収束する。
$a_n \to a$ とする。任意の $\varepsilon > 0$ に対し、ある $N$ が存在して $n > N \implies |a_n - a| < \varepsilon$ となる。 部分列 $\{a_{n_k}\}$ を考える。$n_k$ は $k$ について狭義単調増加するので、$k \to \infty$ のとき $n_k \to \infty$ である。 よって、上記の $N$ に対して、十分大きな $K$ をとれば、$k > K \implies n_k > N$ とできる。 したがって、$k > K$ ならば $|a_{n_k} - a| < \varepsilon$ が成り立つ。これは $\{a_{n_k}\}$ が $a$ に収束することを意味する。
■

この定理の対偶をとると、収束しないことの判定に使える強力なツールが得られる。

系 (収束しないことの判定法)
数列 $\{a_n\}$ が、異なる極限値に収束する2つの部分列を持つならば、$\{a_n\}$ は収束しない。
例：$\{ (-1)^n \}$ が収束しないことの証明
先に見たように、$\{ (-1)^n \}$ は部分列として $\{1, 1, \dots\}$ と $\{-1, -1, \dots\}$ を持つ。 前者は $1$ に、後者は $-1$ に収束する。 異なる極限値を持つ部分列が存在するので、元の数列 $\{ (-1)^n \}$ は収束しない。
定理 (Bolzano–Weierstrass)
有界な数列は、必ず収束する部分列を持つ。
数列 $\{x_n\}$ は有界であるとする。証明のために「峰」という概念を導入する。

峰 (peak): 項 $x_p$ が峰であるとは、その項より先にあるすべての項がそれより小さいか等しいこと、すなわち $\forall k > p, x_k \le x_p$ が成り立つことをいう。

峰であるような添え字 $p$ の集合を $P$ とする。
  • Case 1: $P$ が無限集合の場合
    $P$ の元を小さい順に $p_1 < p_2 < p_3 < \dots$ とする。峰の定義から、$x_{p_1} \ge x_{p_2} \ge x_{p_3} \ge \dots$ となる。 したがって、部分列 $\{x_{p_k}\}$ は単調減少である。$\{x_n\}$ は有界なので、この部分列も有界である。 有界な単調減少列は収束するので、この場合は収束部分列が見つかった。
  • Case 2: $P$ が有限集合の場合
    $P$ が空集合であるか、有限集合であるとする。$N_0 = \max P$ とする（$P$が空なら $N_0=0$ とする）。 $n_1 > N_0$ なる任意の $n_1$ をとる。$n_1 \notin P$ なので、$n_1$ は峰ではない。 したがって、ある $n_2 > n_1$ が存在して $x_{n_2} > x_{n_1}$ となる。 $n_2 > N_0$ なので $n_2$ も峰ではない。よって、ある $n_3 > n_2$ が存在して $x_{n_3} > x_{n_2}$ となる。 この操作を繰り返すことで、狭義単調増加な部分列 $x_{n_1} < x_{n_2} < x_{n_3} < \dots$ を構成できる。 この部分列は単調増加であり、元の列が有界なので、この部分列も有界である。 したがって、この場合も収束部分列が存在する。
どちらのケースでも収束部分列の存在が示された。

6. 演習問題

  1. $a_n = \dfrac{1}{n}$ が 0 に収束することを ε–N 論法で示せ。
  2. $a_n = \dfrac{(-1)^n}{n}$ の収束先を調べ，証明せよ。
  3. 命題「非有界な数列は、収束する部分列を持たない」は正しいか？理由とともに述べよ。
  4. 有界でありながら収束しない数列の例を1つ挙げ、それが収束しない理由を述べよ。

7. 演習問題の解答

1. $a_n = \dfrac{1}{n} \to 0$ の証明

示すべきこと：$\forall \varepsilon > 0, \exists N \in \mathbb{N} \text{ s.t. } n > N \implies \left|\dfrac{1}{n} - 0\right| < \varepsilon$.

任意の $\varepsilon > 0$ をとる。 $\left|\dfrac{1}{n} - 0\right| = \dfrac{1}{n}$ なので、示すべきは $\dfrac{1}{n} < \varepsilon$ である。 この不等式は $n > \dfrac{1}{\varepsilon}$ と同値である。 アルキメデスの原理より、$\dfrac{1}{\varepsilon}$ を上回る自然数 $N$ が存在する。 この $N$ をとると、$n > N$ である任意の自然数 $n$ に対して、 $$n > N > \frac{1}{\varepsilon}$$ となり、$\dfrac{1}{n} < \varepsilon$ が成り立つ。 よって、$\left|\dfrac{1}{n} - 0\right| < \varepsilon$ が示された。 ■

2. $a_n = \dfrac{(-1)^n}{n}$ の収束先と証明

収束先： 0

証明： 示すべきこと：$\forall \varepsilon > 0, \exists N \in \mathbb{N} \text{ s.t. } n > N \implies \left|\dfrac{(-1)^n}{n} - 0\right| < \varepsilon$.

任意の $\varepsilon > 0$ をとる。 $$\left|\frac{(-1)^n}{n} - 0\right| = \left|\frac{(-1)^n}{n}\right| = \frac{1}{n}$$ よって、示すべきことは $\dfrac{1}{n} < \varepsilon$ である。 これは問題1と全く同じであり、アルキメデスの原理から $\dfrac{1}{\varepsilon}$ を上回る自然数 $N$ をとれば、$n > N$ なるすべての $n$ について $\dfrac{1}{n} < \varepsilon$ が成立する。 ゆえに $\displaystyle \lim_{n\to\infty} \frac{(-1)^n}{n} = 0$ である。 ■

3. 「非有界な数列は、収束する部分列を持たない」は正しいか？

解答：正しくない。

理由（反例）： 次のような数列 $\{a_n\}$ を考える。 $$ a_n = \begin{cases} n & (n \text{ が奇数のとき}) \\ 0 & (n \text{ が偶数のとき}) \end{cases} $$ すなわち、$\{1, 0, 3, 0, 5, 0, 7, \dots\}$ という数列である。 奇数の項は $1, 3, 5, \dots$ と無限に大きくなるため、この数列 $\{a_n\}$ は非有界である。 しかし、$n_k = 2k$ とおけば、偶数番目の項だけを取り出した部分列 $\{a_{2k}\}$ を作ることができる。 この部分列は $\{0, 0, 0, \dots\}$ であり、明らかに 0 に収束する。 したがって、「非有界な数列」が「収束する部分列」を持つ例が存在するため、この命題は偽である。 ■

4. 有界非収束列の例

例： $a_n = (-1)^n$

理由： まず、すべての $n$ に対して $|a_n| = |(-1)^n| = 1$ なので、この数列は $-1$ と $1$ の間にあり、有界である。 次に、この数列が収束しないことを示す。講義で示したように、この数列は異なる極限値を持つ２つの収束部分列を持つ。

  • $n_k = 2k$ （偶数番目）とすると、部分列 $\{a_{2k}\} = \{1, 1, 1, \dots\}$ は $1$ に収束する。
  • $n_k = 2k-1$ （奇数番目）とすると、部分列 $\{a_{2k-1}\} = \{-1, -1, -1, \dots\}$ は $-1$ に収束する。
もし元の数列 $\{a_n\}$ がある値 $\alpha$ に収束すると仮定すれば、そのすべての部分列も同じ値 $\alpha$ に収束するはずである。しかし、この数列は $1$ と $-1$ という異なる値に収束する部分列を持つため、矛盾する。 したがって、数列 $\{(-1)^n\}$ は収束しない。 ■
