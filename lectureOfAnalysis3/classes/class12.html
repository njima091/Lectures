Class 12: Implicit Differentiation and the Implicit Function Theorem

In many practical situations, relationships between variables are given implicitly through equations rather than explicit functions. For example, the equation of a circle $x^2 + y^2 = r^2$ defines $y$ as a function of $x$ implicitly, but solving for $y$ explicitly gives two branches: $y = \pm\sqrt{r^2 - x^2}$. Today, we'll explore implicit differentiation and the powerful Implicit Function Theorem, which allows us to analyze such relationships systematically.

We begin by recalling implicit differentiation from single-variable calculus. When a curve is defined by an equation $F(x,y) = 0$, we can differentiate both sides with respect to $x$, treating $y$ as a function of $x$. Using the chain rule, we get:

$$\frac{d}{dx}F(x,y) = 0 \Rightarrow \frac{\partial F}{\partial x} + \frac{\partial F}{\partial y}\frac{dy}{dx} = 0$$

Solving for $\frac{dy}{dx}$, we obtain the fundamental formula for implicit differentiation:

$$\frac{dy}{dx} = -\frac{\frac{\partial F}{\partial x}}{\frac{\partial F}{\partial y}} = -\frac{F_x}{F_y}$$

This formula gives us the slope of the tangent line to the curve $F(x,y) = 0$ at any point where $F_y \neq 0$.

Example 12.1: Circle Equation

Question: Find $\frac{dy}{dx}$ for the circle $x^2 + y^2 = 9$ at the point $(1, -2\sqrt{2})$.

Solution:

Let $F(x,y) = x^2 + y^2 - 9 = 0$. We compute the partial derivatives:

$F_x = \frac{\partial F}{\partial x} = 2x$

$F_y = \frac{\partial F}{\partial y} = 2y$

Using the implicit differentiation formula:

$$\frac{dy}{dx} = -\frac{F_x}{F_y} = -\frac{2x}{2y} = -\frac{x}{y}$$

At the point $(1, -2\sqrt{2})$:

$$\frac{dy}{dx} = -\frac{1}{-2\sqrt{2}} = \frac{1}{2\sqrt{2}} = \frac{\sqrt{2}}{4}$$

This is the slope of the tangent line to the circle at the given point.

The Implicit Function Theorem generalizes this idea, providing conditions under which an equation $F(x,y) = 0$ locally defines $y$ as a differentiable function of $x$. More precisely:

Let $F(x,y)$ be a continuously differentiable function on an open set in $\mathbb{R}^2$ containing the point $(a,b)$ where $F(a,b) = 0$. If $\frac{\partial F}{\partial y}(a,b) \neq 0$, then:

  1. There exist open intervals $I$ containing $a$ and $J$ containing $b$, and a unique continuously differentiable function $g: I \rightarrow J$ such that:
    • $g(a) = b$
    • $F(x, g(x)) = 0$ for all $x \in I$
  2. The derivative of $g$ is given by: $g'(x) = -\frac{F_x(x,g(x))}{F_y(x,g(x))}$

In other words, if $F_y(a,b) \neq 0$, then near $(a,b)$, the equation $F(x,y) = 0$ implicitly defines $y$ as a differentiable function of $x$. This function's derivative is given by the same formula we derived earlier.

Figure 12.1: The Implicit Function Theorem guarantees that near a point where $F_y \neq 0$, the level curve $F(x,y) = 0$ can be expressed as a function $y = g(x)$.

The geometric intuition is that when $F_y(a,b) \neq 0$, the level curve $F(x,y) = 0$ does not have a vertical tangent at $(a,b)$. This allows us to solve for $y$ as a function of $x$ locally. The condition $F_y(a,b) \neq 0$ means the partial derivative with respect to $y$ is non-zero, ensuring we can "isolate" $y$.

Example 12.2: Folium of Descartes

Question: Consider the Folium of Descartes given by the equation $x^3 + y^3 = 3xy$. Determine where the Implicit Function Theorem applies and find $\frac{dy}{dx}$ at the point $(2,1)$.

Solution:

Let $F(x,y) = x^3 + y^3 - 3xy = 0$. The partial derivatives are:

$F_x = 3x^2 - 3y$

$F_y = 3y^2 - 3x$

The Implicit Function Theorem applies at points where $F_y \neq 0$, that is, where $3y^2 - 3x \neq 0$ or $y^2 \neq x$. At such points, $y$ can be expressed locally as a function of $x$.

At $(2,1)$, we check:

$F_y(2,1) = 3(1)^2 - 3(2) = 3 - 6 = -3 \neq 0$

Therefore, the Implicit Function Theorem applies at $(2,1)$, and we can compute:

$$\frac{dy}{dx} = -\frac{F_x}{F_y} = -\frac{3x^2 - 3y}{3y^2 - 3x} = -\frac{x^2 - y}{y^2 - x}$$

At the point $(2,1)$:

$$\frac{dy}{dx} = -\frac{2^2 - 1}{1^2 - 2} = -\frac{3}{-1} = 3$$

This tells us that the tangent line to the Folium at $(2,1)$ has a slope of 3.

The Implicit Function Theorem extends naturally to higher dimensions. For example, if we have an equation $F(x,y,z) = 0$ where $F$ is a continuously differentiable function, and if $F_z(a,b,c) \neq 0$ at a point $(a,b,c)$ where $F(a,b,c) = 0$, then near this point, $z$ can be expressed as a function of $x$ and $y$: $z = g(x,y)$. The partial derivatives of $g$ are given by:

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

This higher-dimensional version is particularly useful in many engineering applications, where multiple variables are related through constraint equations.

Example 12.3: 3D Surface

Question: Consider the equation $F(x,y,z) = x^2 + y^2 + z^2 - 16 = 0$, which defines a sphere of radius 4 centered at the origin. Compute $\frac{\partial z}{\partial x}$ and $\frac{\partial z}{\partial y}$ at the point $(3,2,3)$.

Solution:

First, let's verify the point is on the sphere:

$3^2 + 2^2 + 3^2 = 9 + 4 + 9 = 22 \neq 16$

The point doesn't lie on the sphere. Let's use $(3,2,\sqrt{7})$ instead, which does lie on the sphere.

We compute the partial derivatives:

$F_x = 2x$

$F_y = 2y$

$F_z = 2z$

At any point where $z \neq 0$, the Implicit Function Theorem applies. At $(3,2,\sqrt{7})$:

$F_z(3,2,\sqrt{7}) = 2\sqrt{7} \neq 0$

Therefore:

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{2x}{2z} = -\frac{x}{z} = -\frac{3}{\sqrt{7}}$$ $$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{2y}{2z} = -\frac{y}{z} = -\frac{2}{\sqrt{7}}$$

These derivatives describe how $z$ changes locally with respect to small changes in $x$ and $y$ while staying on the sphere.

The Implicit Function Theorem has many important applications:

Applications of the Implicit Function Theorem

  • Geometry: Analyzing level sets, surfaces, and curves without explicitly solving for one variable.
  • Optimization with Constraints: The method of Lagrange multipliers relies on implicit functions.
  • Differential Equations: Proving existence and uniqueness of solutions to certain types of equations.
  • Economics: Analyzing equilibrium conditions and comparative statics in economic models.
  • Engineering Design: Understanding how changes in design parameters affect performance metrics.
Example 12.4: Economic Application

Question: In economics, a Cobb-Douglas production function $F(K,L) = AK^{\alpha}L^{1-\alpha}$ relates output to capital ($K$) and labor ($L$). Market equilibrium requires that the marginal product of labor equals the wage rate $w$: $\frac{\partial F}{\partial L} = w$. Express $L$ in terms of $K$ and $w$ implicitly, and find $\frac{dL}{dK}$ at equilibrium.

Solution:

The marginal product of labor is:

$$\frac{\partial F}{\partial L} = (1-\alpha)AK^{\alpha}L^{-\alpha}$$

The equilibrium condition $\frac{\partial F}{\partial L} = w$ gives:

$$G(K,L,w) = (1-\alpha)AK^{\alpha}L^{-\alpha} - w = 0$$

This equation implicitly defines $L$ as a function of $K$ and $w$. We compute the partial derivatives:

$$G_K = \alpha(1-\alpha)AK^{\alpha-1}L^{-\alpha}$$ $$G_L = -\alpha(1-\alpha)AK^{\alpha}L^{-\alpha-1}$$

By the Implicit Function Theorem (assuming $G_L \neq 0$):

$$\frac{dL}{dK} = -\frac{G_K}{G_L} = -\frac{\alpha(1-\alpha)AK^{\alpha-1}L^{-\alpha}}{-\alpha(1-\alpha)AK^{\alpha}L^{-\alpha-1}} = \frac{L}{K}$$

This tells us that at equilibrium, the percentage change in labor is equal to the percentage change in capital, maintaining the same wage rate. This is a key result in economic growth theory.

It's important to note some limitations of the Implicit Function Theorem:

  1. It only guarantees local solvability. The function $g$ might only be defined on a small interval around the point of interest.
  2. The condition $F_y \neq 0$ is necessary. At points where $F_y = 0$ (corresponding to vertical tangents in the case of curves), the theorem doesn't apply.
  3. The theorem provides existence and differentiability, but not always an explicit formula for the function $g$. In practice, we often work with the derivatives directly without solving for $g$.

In summary, implicit differentiation and the Implicit Function Theorem provide powerful tools for analyzing relationships between variables given by equations rather than explicit functions. The key insight is that under appropriate conditions (non-zero partial derivative with respect to the dependent variable), we can treat one variable as a function of the others locally, and compute its derivatives without explicitly solving for the function.

Key Insights about Implicit Differentiation

  • Implicit differentiation allows us to find derivatives without solving for one variable explicitly.
  • The formula $\frac{dy}{dx} = -\frac{F_x}{F_y}$ works when $F_y \neq 0$.
  • The Implicit Function Theorem guarantees the existence of a local function satisfying the implicit equation.
  • In higher dimensions, similar formulas apply: $\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}$, $\frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$.
  • These techniques are essential for analyzing complex relationships in mathematics, physics, engineering, and economics.
